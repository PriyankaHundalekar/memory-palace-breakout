## üöÄ Getting Started

### Quick Start
1. **Download or Clone**
   ```bash
   git clone <repository-url>
   cd memory-palace-breakout
   ```

2. **Start Local Server** (recommended)
   ```bash
   # Using Python
   python -m http.server 3000
   
   # Using Node.js
   npx serve .
   
   # Using PHP
   php -S localhost:3000
   ```

   Navigate to: `http://localhost:3000/memory-palace-breakout.html`
